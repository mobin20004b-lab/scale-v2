# ESP32 Scale Agent

این ماژول نمونه برای هر ترازو یک دستگاه ESP32 در نظر می‌گیرد و قابلیت‌های زیر را پوشش می‌دهد:

- Enrollment با bootstrap token
- ارسال وزن و telemetry (uptime/model/heartbeat)
- polling فرمان‌ها و ارسال ACK/FAIL
- دریافت تنظیمات چاپگر (TSPL/ESC-POS)
- مدیریت محلی از طریق captive portal
